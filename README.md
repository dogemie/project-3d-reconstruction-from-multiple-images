# 다중 이미지를 통한 3차원 복원 (3D Reconstruction from Multiple Images)

3D Gaussian Splatting(3DGS) 기법을 활용하여 추출된 3차원 객체를 새로운 3D 환경 내에 자연스럽게 자동 배치하는 시스템을 개발하였다. 주성분 분석(PCA)을 통해 환경 데이터 내의 평면 후보군을 탐색 및 검출하고, 배치 시의 안정성, 충돌 여부, 접촉 면접 등을 종합적으로 평가하는 점수 함수 모델을 설계하였다. 이를 통해 2D 동영상으로부터 3D 모델을 생성하고 불필요한 노이즈 포인트를 필터링하는 과정을 거쳐, 사용자의 수동적인 개입을 최소화하면서 객체가 최적의 위치와 자세로 배경 내에 자동 배치되는 파이프라인을 구축하였다.

## 1. 프로젝트 개요

### 1.1 프로젝트 목적 및 동기
 본 프로젝트는 3D Gaussian Splatting(3DGS)으로 재구성된 객체를 새로운 3차원 환경에 자동으로 배치하는 시스템을 개발하는 것을 목표로 한다. 사용자가 객체의 위치와 자세를 직접 지정하지 않아도, 환경의 기하학적 특성을 분석하여 가장 적절한 배치 위치를 자동으로 결정하는 기술을 구현한다.
 최근 3D Gaussian Splatting은 적은 수의 이미지로도 고품질의 3차원 장면을 빠르게 재구성할 수 있어 디지털 트윈, 메타버스, AR/VR 등의 분야에서 활발히 활용되고 있다. 그러나 기존 연구는 객체나 환경을 재구성하는 데 집중되어 있으며, 재구성된 객체를 새로운 공간에 자연스럽게 배치하는 과정은 대부분 사용자의 수동 조작에 의존하고 있다.
 또한 실제 환경에서 객체를 배치하기 위해서는 단순한 위치 이동이 아니라 환경의 구조를 이해하고 객체의 형태와 안정성을 고려하는 과정이 필요하다. 이를 위해 본 프로젝트에서는 Point Cloud 기반 기하 분석, PCA(Principal Component Analysis), Plane Detection, 충돌 검사 등의 기법을 활용한다.
 3D 콘텐츠 제작 과정에서 객체 배치는 반복적으로 수행되는 작업이지만 많은 시간과 수작업이 요구된다. 특히 AR/VR, 게임, 디지털 트윈과 같은 응용 분야에서는 다수의 객체를 다양한 환경에 배치해야 하므로 자동화 기술의 필요성이 증가하고 있다.
 본 프로젝트는 환경의 구조를 분석하여 객체를 자동으로 배치함으로써 콘텐츠 제작 효율을 높이고, 향후 다양한 3D 공간 편집 및 생성 시스템에 활용될 수 있는 기반 기술을 제공하고자 한다.

### 1.2 프로젝트 목표
 3D Gaussian Splatting으로 재구성된 객체를 입력받아 새로운 3차원 환경 내에서 가장 적절한 위치와 자세를 자동으로 결정하고 배치하는 시스템을 개발하는 것을 최종 목표로 한다.
이를 통해 사용자의 직접적인 위치 지정 없이도 자연스럽고 안정적인 3D 장면을 생성할 수 있는 자동 배치 파이프라인을 구축한다.
 1) 객체의 기하학적 특성 분석
	- 객체 Point Cloud를 추출
	- PCA를 이용하여 객체의 주축(Principal Axis) 추정
	- 객체의 방향 및 자세 정보 획득
2) 환경 구조 분석
	- 환경 Point Cloud 생성
	- Plane Detection을 통해 바닥, 테이블, 벤치 등 배치 가능한 평면 영역 탐색
	후보 배치 영역 추출
3) 자동 배치 알고리즘 설계
	- 객체를 각 후보 영역에 가상 배치
	- 접촉 면적, 안정성, 충돌 여부 등을 평가
	- 배치 점수(Placement Score) 계산
4) 최적 위치 선정 및 배치
	- 후보 위치들 중 최고 점수를 갖는 위치 선택
	- 객체의 위치 및 자세 자동 결정
	- 최종 3D 장면 생성
5) 시스템 검증
	- 다양한 객체 및 환경 조합에 대해 실험 수행
	- 자동 배치 결과의 안정성과 자연스러움 평가
	- 실제 활용 가능성 검증

## 2. 프로젝트 내용

### 2.1 프로젝트 제반 사항
 본 프로젝트는 3D Gaussian Splatting(3DGS)으로 재구성된 객체를 새로운 3차원 환경에 자동으로 배치하는 시스템 개발을 목표로 한다. 이를 위해 객체와 환경을 3차원 데이터로 표현하고 분석할 수 있는 기술이 필요하다.
 먼저 다수의 이미지로부터 3차원 장면을 재구성하기 위해 COLMAP 기반 Structure-from-Motion(SfM) 및 Multi-View Stereo(MVS) 기술을 활용하며, 이를 통해 생성된 데이터를 기반으로 3D Gaussian Splatting 모델을 구축한다. 또한 객체와 환경의 기하학적 특성을 분석하기 위해 Point Cloud 처리 기술, PCA(Principal Component Analysis), Plane Detection, Voxel Downsampling 등의 알고리즘이 필요하다.
 자동 배치 과정에서는 객체의 방향 추정, 환경 내 배치 가능 영역 탐색, 충돌 검사, 안정성 평가 등의 기능이 요구되며, 이를 위해 Open3D, NumPy, PyTorch 등의 라이브러리를 활용하여 시스템을 구현한다.

### 2.2 관련 기술 사전 조사
 본 프로젝트 수행에 앞서 관련 논문, 기술 문서 및 공개 소프트웨어를 조사하여 기초 기술을 학습하였다.
(1) 3D Gaussian Splatting
 2023년 발표된 3D Gaussian Splatting 연구를 중심으로 학습하였다. 해당 기술은 기존 NeRF 기반 방식보다 빠른 학습 속도와 실시간 렌더링 성능을 제공하며, 적은 수의 이미지로도 고품질 3차원 장면을 재구성할 수 있다는 장점을 가진다.
(2) Point Cloud Processing
 객체와 환경의 공간 정보를 분석하기 위해 Point Cloud 자료구조와 처리 기법을 조사하였다. 특히 Point Cloud Downsampling, Surface Normal Estimation, Spatial Search 등의 기법을 학습하였다.
(3) PCA 기반 자세 추정
 객체 Point Cloud의 주축을 추정하기 위해 PCA 알고리즘을 조사하였다. PCA는 데이터의 분산이 가장 큰 방향을 추출하여 객체의 대표 방향을 계산할 수 있으며, 이를 통해 자동 배치 시 객체의 초기 자세를 결정할 수 있다.
(4) Plane Detection
 환경 내에서 객체가 배치될 수 있는 영역을 찾기 위해 RANSAC 기반 Plane Detection 기법을 조사하였다. 이를 통해 바닥, 책상, 벤치 등의 평면 영역을 자동으로 검출할 수 있음을 확인하였다.
(5) Collision Detection
 배치된 객체가 환경과 겹치는 문제를 방지하기 위해 Bounding Box 및 Point Cloud 기반 충돌 검사 기법을 조사하였다.

### 2.3 현재 기술의 한계성과 연구 개발 동향
 최근 3D Gaussian Splatting 기술은 빠르게 발전하고 있으며, 객체 생성 및 장면 재구성 분야에서 높은 성능을 보이고 있다. 그러나 대부분의 연구는 장면을 재구성하거나 렌더링하는 데 초점을 맞추고 있으며, 재구성된 객체를 새로운 환경에 자동으로 배치하는 기능은 상대적으로 부족한 상황이다.
 현재 상용 3D 편집 소프트웨어 및 콘텐츠 제작 도구에서는 사용자가 직접 객체의 위치와 회전을 조정해야 하는 경우가 많으며, 자동 배치 기능이 존재하더라도 단순한 규칙 기반 방식에 의존하는 경우가 대부분이다.
 최근 연구에서는 Scene Understanding, Spatial Reasoning, 3D Scene Composition 등을 활용하여 객체와 환경 간의 관계를 이해하고 자동으로 배치하려는 시도가 증가하고 있다. 특히 Point Cloud 분석, 물리 기반 시뮬레이션, AI 기반 Scene Generation 기술을 결합하여 보다 자연스러운 객체 배치를 수행하려는 연구가 활발하게 진행되고 있다.

### 2.4 연구 개발 계획
 본 프로젝트에서는 3DGS 기반 객체 자동 배치 시스템을 구현하기 위해 다음과 같은 단계로 연구개발을 수행한다.
 
1) 객체 및 환경 데이터 구축
	- 다중 시점 이미지 촬영
	- COLMAP을 이용한 카메라 포즈 추정
	- 3D Gaussian Splatting 모델 생성
	- 객체 및 환경 Point Cloud 추출

2) 객체 특성 분석
	- Point Cloud 전처리
	- Voxel Downsampling 수행
	- PCA 기반 객체 주축 추정
	- 객체 방향 정보 계산

3) 환경 구조 분석
	- Plane Detection 수행
	- 배치 가능한 평면 후보 탐색
	- 후보 영역의 기하학적 특성 분석

4) 자동 배치 알고리즘 개발
	- 객체를 후보 평면에 가상 배치
	- 접촉 면적 계산
	- 충돌 여부 검사
	- 무게중심 기반 안정성 평가
	- Placement Score 계산

5) 최적 위치 선정 및 결과 생성
	- 후보 영역 중 최고 점수 위치 선택
	- 객체 위치 및 자세 자동 결정
	- 최종 3D 장면 생성 및 시각화

6) 성능 평가
	- 다양한 객체 및 환경 조합 실험
	- 배치 안정성 및 자연스러움 평가
	- 자동 배치 성공률 분석

### 3.2 적용 기술 소개 및 적용 방법
1) 3D Gaussian Splatting
 본 프로젝트에서는 객체와 환경을 3차원 공간으로 재구성하기 위해 3D Gaussian Splatting을 활용하였다. 사용자가 촬영한 다중 시점 이미지를 입력으로 사용하며, COLMAP을 통해 카메라 위치와 자세를 추정한 후 Gaussian 기반 장면 표현을 생성하였다. 이를 통해 객체와 환경에 대한 고품질 3차원 데이터를 확보할 수 있었다.

2) Point Cloud 기반 기하 분석
 3DGS로 생성된 데이터를 Point Cloud 형태로 변환하여 기하학적 분석을 수행하였다. Point Cloud는 객체의 위치와 형상을 직접적으로 표현할 수 있어 배치 알고리즘 개발에 적합하였다. 또한 계산량 감소를 위해 Voxel Downsampling을 적용하여 포인트 수를 줄이고 처리 속도를 향상시켰다.

3) PCA 기반 객체 방향 추정
 자동 배치 과정에서 객체의 방향을 결정하기 위해 PCA를 활용하였다. 객체 Point Cloud의 공분산 행렬을 계산하고 고유벡터를 추출하여 분산이 가장 큰 방향을 객체의 주축으로 정의하였다. 이를 통해 객체의 초기 자세를 자동으로 추정할 수 있었으며, 수동 방향 설정 없이 배치가 가능하도록 구현하였다.

4) Plane Detection
 환경 내 배치 가능한 영역을 찾기 위해 RANSAC 기반 Plane Detection 알고리즘을 적용하였다. 환경 Point Cloud에서 평면을 반복적으로 탐색하여 바닥, 책상, 벤치와 같은 후보 영역을 추출하였다. 검출된 평면은 이후 자동 배치 알고리즘의 후보 위치로 활용하였다.

5) Placement Scoring 알고리즘
 본 프로젝트의 핵심 기술은 객체 자동 배치를 위한 Placement Scoring 알고리즘이다.
후보 평면마다 객체를 가상 배치한 뒤 “평면과의 접촉 정도, 환경과의 충돌 여부, 객체의 안정성, 객체의 배치 가능 면적”의 요소들을 종합하여 점수를 계산하고 가장 높은 점수를 갖는 위치를 최종 배치 위치로 선정하였다. 이를 통해 단순히 가장 큰 평면을 선택하는 것이 아니라 객체와 환경의 특성을 고려한 배치가 가능하게 하였다.

### 3.3 기술적 어려움 및 해결 과정
1) 객체 방향 추정 문제
 초기에는 객체가 어떤 방향으로 저장되어 있는지 알 수 없다는 문제가 존재하였다. 예를 들어 인형 데이터의 경우 촬영 방식에 따라 누워 있는 상태로 저장될 수 있으며, 이를 그대로 배치할 경우 비정상적인 결과가 발생하였다. 이를 해결하기 위해 PCA를 이용하여 객체의 주축을 계산하고 자동으로 방향을 정렬하는 방법을 적용하였다. 그 결과 객체의 초기 자세를 자동으로 추정할 수 있었으며 다양한 객체에 대해 일관된 배치가 가능해졌다.

2) 대용량 Point Cloud 처리 문제
 3DGS에서 생성된 Point Cloud는 수십만 개 이상의 포인트를 포함하고 있어 Plane Detection 및 충돌 검사 수행 시 계산량이 크게 증가하였다. 이를 해결하기 위해 Voxel Downsampling을 적용하여 Point Cloud 밀도를 조절하였다. 이를 통해 객체 형상의 특징은 유지하면서도 처리 시간을 크게 단축할 수 있었다.

3) 적절한 배치 위치 선정 문제
 환경 내에서 검출된 평면 중 가장 넓은 평면이 항상 최적의 배치 위치가 되는 것은 아니었다. 예를 들어 벤치가 존재하는 환경에서는 바닥보다 벤치 위에 객체를 배치하는 것이 더욱 자연스러운 결과를 생성할 수 있다. 이를 해결하기 위해 단순 평면 검출이 아닌 Placement Scoring 방식을 설계하였다. 접촉 면적, 안정성, 충돌 여부 등을 종합적으로 평가하여 가장 자연스러운 위치를 선택하도록 구현하였다.

4) 환경 데이터 정렬 문제
 촬영된 환경 데이터는 항상 수평 상태로 재구성되지 않았으며, 이로 인해 Plane Detection 결과가 불안정하게 나타나는 문제가 발생하였다. 이를 해결하기 위해 검출된 평면의 법선 벡터를 이용하여 환경을 정렬하고, 수평 기준을 자동으로 설정하는 방식을 적용하였다. 이를 통해 다양한 촬영 환경에서도 안정적인 평면 검출이 가능해졌다.

## 4. 결론 및 기대효과

### 4.1 프로젝트 결과 및 소감
 본 프로젝트를 수행하면서 단순히 3D 데이터를 생성하는 것에서 나아가, 객체와 환경의 공간적 관계를 분석하고 이를 바탕으로 자동 배치를 수행하는 과정의 중요성을 경험할 수 있었다. 특히 3D Gaussian Splatting, Point Cloud Processing, PCA, Plane Detection 등 다양한 컴퓨터 비전 및 기하학 기반 기술을 실제 시스템으로 구현하면서 이론과 실제 개발 간의 차이를 체감할 수 있었다.
 프로젝트 초기에는 Docker를 이용한 환경 설정, 3DGS를 이용한 객체 재구성에 집중하였으나, 실제로는 객체의 방향 추정, 환경 분석, 충돌 검사, 배치 평가와 같은 추가적인 문제들을 해결해야 자연스러운 결과를 얻을 수 있다는 점을 확인하였다. 이러한 과정을 통해 문제를 세분화하여 분석하고, 여러 기술을 통합하여 하나의 시스템으로 구현하는 경험을 쌓을 수 있었다.

### 4.2 활용 방안 및 기대 효과
 본 프로젝트에서 개발한 자동 배치 기술은 다양한 3차원 콘텐츠 제작 분야에 활용될 수 있다. 첫째, AR/VR 및 메타버스 환경에서 사용자가 생성한 객체를 자동으로 배치하여 콘텐츠 제작 과정을 단순화할 수 있다. 둘째, 디지털 트윈 환경에서 설비, 가구, 장비 등의 객체를 가상 공간에 자동으로 배치함으로써 공간 구성 및 시뮬레이션 작업의 효율성을 향상시킬 수 있다. 셋째, 게임 및 3D 콘텐츠 제작 분야에서 반복적으로 수행되는 객체 배치 작업을 자동화하여 제작 시간과 비용을 절감할 수 있다.
 또한 본 프로젝트는 단순한 객체 이동이 아닌 환경의 기하학적 구조를 분석하여 최적의 위치를 선택하는 방식을 적용하였기 때문에 향후 지능형 3D 공간 편집 기술의 기반 연구로도 활용될 수 있을 것으로 기대된다.

### 4.3 향후 계획
 현재 시스템은 평면 기반의 후보 영역을 탐색하여 객체를 배치하는 방식으로 구현되어 있다. 향후에는 단순한 기하학적 정보뿐만 아니라 객체와 환경의 의미적 관계(Semantic Information)를 활용하여 보다 자연스러운 배치를 수행할 수 있도록 연구를 확장할 계획이다.
 예를 들어 인형은 의자나 벤치 위에, 컵은 책상 위에 배치하는 등 객체의 종류와 주변 환경을 함께 고려하는 Scene Understanding 기술을 적용할 수 있을 것으로 기대된다.
 또한 물리 엔진을 활용한 시뮬레이션을 도입하여 배치 안정성을 보다 정교하게 평가하고, 다양한 객체와 복잡한 환경에서도 동작할 수 있는 범용 자동 배치 시스템으로 발전시키고자 한다.
 장기적으로는 생성형 AI와 결합하여 사용자가 원하는 장면을 입력하면 객체 생성부터 공간 배치까지 자동으로 수행하는 지능형 3D 콘텐츠 생성 시스템으로 확장하는 것을 목표로 한다.
